What is Language Notes?

Language notes is an online SRS (Spaced Repetition System) flashcard program used to help you learn a new language. Adding SRS system to flashcards makes learning more efficient by not introducing already learned cards to your face every other minute.

How do I use it?

At start you might want to create yourself a list where you can add your own cards. A list can be created in My lists tab in navigation bar.

Now you probably want to add cards to your list. That can be done by clicking the list name which opens up your list. There you can add cards. When you are finished adding cards. You can study them in Study mode by pressing Study in your list. Remember, you can add, edit and remove cards as you please.

If you know a friend using Language Notes, you can add him/her by searching their username. Searching a user opens their profile page. There you can find a follow button and their lists and languages.

You can also add yourself languages that you are learning. You can add those by going to your profile page and click Edit profile.
